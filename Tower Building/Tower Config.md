The tower config controls how the game interprets a tower.
# Schema
- `TowerID`: The shorter version of the tower's name. It must be a `string`
- `TowerName`: The name of the tower. It must be a `string`
- `Spawn`: The spawn part inside of the tower
- `Difficulty`: The difficulty of the tower in number form
- `BadgeId`: The id of The badge that is awarded when the tower is beaten
