A part that deals repetitive damage at a set interval as long as a player is touching it.
> [!attention]
> Only applies to the local player to prevent health update conflicts.
# Config
- `Damage`: How much damage is dealt at each interval. Default is `3`
- `Rate`: How long before damage is dealt again. Default is `0.5`
# Tags
- `PoisonPart`
